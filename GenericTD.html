Generic Tower Defense

Introduction to the project

Generic Tower Defense is a classic tower defense game project developed as a part of a game development course. The objective of the game is to strategically place towers to defend against waves of enemies.

Gameplay

The gameplay consists of various levels, each with increasing difficulty. Players earn currency by defeating enemies, which can be used to purchase and upgrade different types of towers.

Features

  • Multiple tower types with unique abilities
  • Upgradable towers
  • Diverse enemy types
  • Special abilities and power-ups
  • Leaderboards to compete with other players

Development Process

The development process involved designing game mechanics, creating assets, coding gameplay features, and testing for bugs and balance.
